Glosario de TI 2

1.) N


  • Node.js

    Node.js es un entorno de ejecución de JavaScript que permite ejecutar código JavaScript en el servidor. Basado en el motor V8 de Google Chrome, facilita el desarrollo de aplicaciones de red escalables y rápidas mediante un modelo de E/S no bloqueante y orientado a eventos.

    • Nombre de Dominio

      Un nombre de dominio es la dirección única y legible para humanos que se utiliza para identificar un sitio web en lugar de utilizar su dirección IP. Por ejemplo, "www.ejemplo.com" es un nombre de dominio que se traduce a una dirección IP para localizar un servidor específico en Internet.

      • 2.) O


      • Open Source

        Open Source o código abierto hace referencia a un modelo de desarrollo de software en el que el código fuente está disponible para el público, permitiendo su modificación, mejora y distribución. Ejemplos populares incluyen Linux, Apache y WordPress.

        • Overflow

          En programación, un "overflow" ocurre cuando una operación aritmética da como resultado un valor que supera la capacidad máxima de almacenamiento del tipo de dato utilizado, lo que puede desencadenar errores o comportamientos inesperados.

          • 3.) P

          • PHP

            PHP es un lenguaje de programación del lado del servidor diseñado para el desarrollo web. Permite crear páginas web dinámicas y es especialmente utilizado para trabajar con bases de datos y servidores.

            • Protocolo

              Un protocolo es un conjunto de reglas y estándares que determinan cómo los dispositivos deben comunicarse en una red. Algunos protocolos comunes son HTTP, HTTPS, FTP y TCP/IP.

              • 4.) Q

              • Query

                Una query o consulta es una solicitud de información realizada a una base de datos. Por ejemplo, en SQL, una consulta SELECT se utiliza para recuperar datos específicos de una tabla.

                • QoS

                  QoS (Quality of Service) es un conjunto de tecnologías utilizadas para gestionar y priorizar el tráfico de red, garantizando un nivel específico de rendimiento y minimizando la latencia o pérdida de datos en aplicaciones críticas.

                  • 5.) R

                  • React

                    React es una biblioteca de JavaScript utilizada para construir interfaces de usuario interactivas basadas en componentes. Fue desarrollada por Facebook y es ampliamente utilizada para crear aplicaciones de una sola página.

                    • REST

                      REST (Representational State Transfer) es un estilo arquitectónico que utiliza HTTP para crear servicios web ligeros y escalables. Los servicios RESTful permiten a las aplicaciones comunicarse mediante solicitudes HTTP estándar como GET, POST, PUT y DELETE.

                      • 6.) S

                      • SEO

                        SEO (Search Engine Optimization) es el proceso de optimizar un sitio web para mejorar su visibilidad en los resultados de búsqueda orgánicos de los motores de búsqueda. Incluye técnicas como la optimización de palabras clave, enlaces y estructura del sitio.

                        • SQL

                          SQL (Structured Query Language) es un lenguaje estándar para gestionar y manipular bases de datos relacionales. Permite ejecutar consultas, insertar, actualizar y eliminar datos en una base de datos.

                          • 7.) T

                          • TCP/IP

                            TCP/IP (Transmission Control Protocol/Internet Protocol) es un conjunto de protocolos fundamentales que permiten la comunicación entre dispositivos en Internet. TCP garantiza la entrega confiable de datos, mientras que IP gestiona las direcciones y el enrutamiento.

                            • Token

                              Un token es una representación digital de un dato o valor en sistemas de seguridad y autenticación. Puede utilizarse para validar identidades, acceder a recursos o realizar transacciones.

                              • 8.) U

                              • URL

                                URL (Uniform Resource Locator) es la dirección única que identifica un recurso específico en Internet, como una página web o un archivo. Ejemplo: https://www.ejemplo.com.

                                • 9.) V

                                • VPN

                                  Una VPN (Virtual Private Network) es una tecnología que permite crear una conexión segura y encriptada entre un dispositivo y una red, protegiendo los datos transmitidos y ocultando la identidad del usuario.

                                  • 10.) W

                                  • WebSocket

                                    WebSocket es un protocolo que permite establecer una conexión bidireccional en tiempo real entre un cliente y un servidor, facilitando la transmisión de datos de forma continua y eficiente.

                                    • 11.) X

                                    • XML

                                      XML (Extensible Markup Language) es un lenguaje de marcado utilizado para estructurar y transportar datos de manera legible para humanos y máquinas. Es ampliamente usado en aplicaciones web y sistemas de intercambio de información.

                                      • 12.) Y

                                      • YAML

                                        YAML (YAML Ain't Markup Language) es un formato de serialización de datos legible para humanos que se utiliza para configurar archivos en aplicaciones y sistemas de software.

                                        • 13.) Z

                                        • Zip

                                          ZIP es un formato de compresión de archivos que reduce el tamaño de uno o varios archivos combinándolos en un único archivo comprimido. Es útil para ahorrar espacio y facilitar la transferencia de archivos.
