GLOSARIO A-M

A B C D E F G H I J K L M N-Z ➡️

A

Algoritmo Genético: Técnica de optimización basada en la evolución natural.

API Gateway: Puerta de enlace para gestionar múltiples servicios API.

AJAX: Conjunto de técnicas para crear aplicaciones web interactivas.

ATM: Asynchronous Transfer Mode, tecnología de redes de alta velocidad.

B

Blockchain: Sistema de registro descentralizado y seguro.

Bug: Error o falla en un programa.

Big Data: Conjunto masivo de datos que requiere técnicas avanzadas para su análisis.

Botnet: Red de dispositivos infectados controlados de forma remota.

C

CLI: Interfaz de Línea de Comandos, modo de interacción textual con el sistema.

Cookie: Fragmento de datos almacenado en el navegador del usuario.

Checksum: Valor calculado para verificar la integridad de datos.

CRM: Gestión de Relaciones con el Cliente mediante software.

D

Data Mining: Proceso de extracción de patrones en grandes conjuntos de datos.

Daemon: Proceso que se ejecuta en segundo plano en sistemas Unix/Linux.

Dock: Barra que agrupa accesos directos en el sistema operativo.

Data Warehouse: Almacén centralizado para datos empresariales.

E

Extranet: Red privada que permite acceso a usuarios externos.

Emulación: Imitación del comportamiento de un sistema mediante software.

Ethernet Switch: Dispositivo que gestiona el tráfico de red local.

Exploit: Código diseñado para aprovechar vulnerabilidades en software.

F

Fragmentación: División de archivos en sectores no contiguos del disco.

Front-End: Parte visible y accesible por el usuario en una aplicación web.

FTP: Protocolo de Transferencia de Archivos.

Firewall: Barrera de seguridad entre redes.

G

Gateway: Dispositivo que conecta redes diferentes.

Gopher: Protocolo de búsqueda y recuperación de documentos.

GUI Toolkit: Conjunto de herramientas para crear interfaces gráficas.

Gigahertz (GHz): Unidad de frecuencia de los procesadores.

H

Hotfix: Actualización rápida para corregir errores críticos.

Hashing: Proceso de conversión de datos a un valor fijo mediante una función hash.

Hypervisor: Software que permite ejecutar máquinas virtuales.

Heurística: Técnica para resolver problemas mediante aproximaciones.

I

Indexación: Organización de datos para optimizar su búsqueda.

ISA: Arquitectura de conjunto de instrucciones de un procesador.

IPv6: Nueva versión del Protocolo de Internet con más direcciones.

Incremental Backup: Copia de seguridad que solo guarda los cambios desde el último backup.

J

JIT: Compilación Just-In-Time, método para ejecutar código a medida que se necesita.

Java Applet: Aplicación pequeña escrita en Java y embebida en una página web.

JSP: Java Server Pages, tecnología para crear contenido dinámico en servidores web.

JDBC: Interfaz de conexión entre Java y bases de datos.

L

Load Balancer: Dispositivo o software que distribuye el tráfico de red.

Log File: Archivo que registra eventos y actividades del sistema.

Loopback: Dirección IP utilizada para pruebas internas (127.0.0.1).

Low-Level Formatting: Proceso de crear sectores físicos en un disco.

M

Middleware: Software que conecta aplicaciones o servicios.

Multithreading: Técnica que permite ejecutar múltiples hilos de un programa al mismo tiempo.

Mesh Network: Red en la que los dispositivos están interconectados entre sí.

Microkernel: Núcleo mínimo del sistema operativo que gestiona funciones esenciales.
