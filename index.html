Glosario de Términos - Informáticos (A-M)

A B C D E F G H I J K L M N-Z ➡️

A

Algoritmo: Conjunto de instrucciones para resolver un problema.

Antivirus: Software diseñado para detectar y eliminar virus informáticos.

API: Interfaz de Programación de Aplicaciones, conjunto de herramientas para construir software.

Archivo: Conjunto de datos almacenados como una unidad.

B

BIOS: Sistema básico de entrada/salida que inicializa el hardware.

Bluetooth: Tecnología inalámbrica de corto alcance.

Backup: Copia de seguridad de datos.

Bit: Unidad mínima de información en computación (0 o 1).

C

CPU: Unidad Central de Procesamiento, cerebro de la computadora.

Cache: Memoria rápida que almacena datos usados frecuentemente.

Compilador: Programa que traduce código fuente a lenguaje máquina.

Cloud Computing: Computación en la nube, servicios a través de Internet.

D

Driver: Software que permite al sistema operativo interactuar con hardware.

DNS: Sistema de Nombres de Dominio, traduce URLs a direcciones IP.

Debugging: Proceso de identificar y corregir errores en software.

Disco Duro: Dispositivo de almacenamiento de datos no volátil.

E

Encriptación: Proceso de codificar información para protegerla.

Extensión: Sufijo de un archivo que indica su formato (.txt, .jpg).

Ethernet: Tecnología para redes cableadas.

EEPROM: Memoria de solo lectura programable y borrable eléctricamente.

F

Firewall: Sistema de seguridad que controla el tráfico de red.

Firmware: Software integrado en hardware.

Fibra Óptica: Medio de transmisión de datos usando luz.

FAT: Tabla de asignación de archivos, sistema de organización en discos.

G

GPU: Unidad de Procesamiento Gráfico.

Gigabyte: Unidad de almacenamiento (aproximadamente 1,000 megabytes).

GUI: Interfaz Gráfica de Usuario.

GPS: Sistema de Posicionamiento Global.

H

HTML: Lenguaje de marcado para crear páginas web.

HTTP: Protocolo de transferencia de hipertexto.

Hardware: Componentes físicos de una computadora.

Hub: Dispositivo que conecta múltiples equipos en una red.

I

IP: Protocolo de Internet, dirección única de un dispositivo en red.

IoT: Internet de las Cosas, conexión de dispositivos cotidianos a Internet.

Inteligencia Artificial: Simulación de inteligencia humana en máquinas.

IDE: Entorno de Desarrollo Integrado para programación.

J

JavaScript: Lenguaje de programación para desarrollo web.

Java: Lenguaje de programación orientado a objetos.

Joystick: Dispositivo de entrada para juegos.

JSON: Formato ligero de intercambio de datos.

K

Kernel: Núcleo del sistema operativo que gestiona recursos.

Kbps: Kilobits por segundo, medida de velocidad de transferencia.

Keylogger: Software malicioso que registra pulsaciones de teclado.

Kilobyte: Unidad de almacenamiento (1,024 bytes).

L

LAN: Red de Área Local.

Linux: Sistema operativo de código abierto.

Latencia: Tiempo de respuesta en una red.

Lenguaje de Programación: Conjunto de instrucciones para crear software.

M

Malware: Software malicioso.

Motherboard: Placa base que conecta componentes de una computadora.

Megabyte: Unidad de almacenamiento (1,024 kilobytes).

Microprocesador: Circuito integrado que ejecuta instrucciones.
