Glosario de Términos - Informáticos (N-Z)

⬅️ A-M N O P Q R S T U V W X Y Z

N

Node.js: Entorno de ejecución para JavaScript del lado del servidor.

NAS: Almacenamiento conectado a red, dispositivo de almacenamiento dedicado.

Navegador: Software para acceder a la web (Chrome, Firefox, Edge).

Nube: Servicios de computación ofrecidos a través de Internet.

O

Overclocking: Aumento de la velocidad de reloj de un componente de hardware.

Open Source: Software de código abierto disponible para modificación.

OCR: Reconocimiento óptico de caracteres para digitalizar texto.

OS: Sistema Operativo, software que gestiona recursos del hardware.

P

Python: Lenguaje de programación de alto nivel y propósito general.

Phishing: Ataque para obtener información confidencial mediante engaño.

Puerto: Interfaz para conectar dispositivos externos.

Proxy: Servidor intermediario entre un usuario e Internet.

Q

Query: Consulta a una base de datos para obtener información.

Qwerty: Distribución estándar de teclados en inglés.

Quantum Computing: Computación cuántica que usa bits cuánticos (qubits).

QuickTime: Framework multimedia de Apple.

R

RAM: Memoria de acceso aleatorio para almacenamiento temporal.

Router: Dispositivo que dirige tráfico entre redes.

RAID: Conjunto redundante de discos independientes para almacenamiento.

ROM: Memoria de solo lectura que almacena firmware.

S

SSD: Disco de estado sólido, más rápido que los discos duros tradicionales.

SQL: Lenguaje de consulta estructurado para bases de datos.

Servidor: Computadora que provee servicios a otras en una red.

SSL: Protocolo de seguridad para conexiones encriptadas.

T

TCP/IP: Protocolos fundamentales para Internet.

Tarjeta Gráfica: Componente que procesa y muestra imágenes.

Terminal: Interfaz de línea de comandos.

Thunderbolt: Interfaz de hardware para conexión de periféricos.

U

USB: Puerto universal para conectar dispositivos.

URL: Localizador uniforme de recursos (dirección web).

UEFI: Interfaz moderna que reemplaza al BIOS.

UPS: Sistema de alimentación ininterrumpida para cortes de energía.

V

VPN: Red privada virtual para conexiones seguras.

Virtualización: Creación de versiones virtuales de recursos.

Virus: Software malicioso que se replica a sí mismo.

VR: Realidad virtual, entorno simulado por computadora.

W

Wi-Fi: Tecnología para redes inalámbricas.

WAN: Red de área amplia que cubre grandes distancias.

Webcam: Cámara para transmitir video por Internet.

Windows: Sistema operativo desarrollado por Microsoft.

X

XML: Lenguaje de marcado extensible para almacenar datos.

XSS: Cross-site scripting, vulnerabilidad de seguridad web.

X86: Arquitectura de procesadores común en PCs.

XAMPP: Paquete de software para servidores web locales.

Y

Yottabyte: Unidad de almacenamiento (1 billón de terabytes).

YAML: Lenguaje de serialización de datos legible por humanos.

YouTube API: Interfaz para integrar YouTube en aplicaciones.

Y2K: Problema del año 2000 en sistemas informáticos.

Z

ZIP: Formato de compresión de archivos.

Zettabyte: Unidad de almacenamiento (1,000 exabytes).

Zócalo: Conector para instalar procesadores en placas base.

Zero-Day: Vulnerabilidad desconocida sin parches disponibles.
