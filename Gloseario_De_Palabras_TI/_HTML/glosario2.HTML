GLOSARIO N-Z

⬅️ A-M N O P Q R S T U V W X Y Z

N

Namespace: Espacio de nombres para organizar identificadores en programación.

NAT: Traducción de direcciones de red, permite compartir una IP pública.

Network Protocol: Conjunto de reglas para la comunicación en redes.

Node: Punto de conexión en una red o estructura de datos.

O

Opcode: Código que indica una operación en lenguaje ensamblador.

OpenCL: Framework para computación paralela en GPU y CPU.

Overflow: Situación donde un valor excede la capacidad de almacenamiento.

ORM: Mapeo Objeto-Relacional para interactuar con bases de datos.

P

Packet: Unidad de datos enviada a través de una red.

Patch: Actualización para corregir errores o vulnerabilidades en software.

Payload: Parte útil de un paquete de datos.

Ping: Comando para verificar la conexión a una dirección IP.

Q

Queue: Estructura de datos que opera en modo FIFO (First In, First Out).

QoS: Calidad de servicio en redes para priorizar ciertos paquetes.

Quantum Encryption: Encriptación basada en principios de mecánica cuántica.

Query Optimization: Mejora del rendimiento de consultas en bases de datos.

R

Repository: Almacén de archivos y código fuente.

RSA: Algoritmo de encriptación asimétrica.

Rollback: Reversión de una transacción o proceso.

RPC: Llamada a Procedimiento Remoto para comunicación entre sistemas.

S

Sandbox: Entorno aislado para ejecutar aplicaciones de forma segura.

SaaS: Software como Servicio, aplicaciones disponibles en la nube.

Scalability: Capacidad de un sistema para crecer sin perder rendimiento.

Socket: Punto final para la comunicación en red.

T

Token: Identificador que autentica a un usuario o transacción.

TTL: Tiempo de vida de un paquete de red antes de ser descartado.

Tunneling: Proceso de encapsular un protocolo dentro de otro.

Telemetry: Recolección de datos a distancia.

U

Unix: Sistema operativo multitarea y multiusuario.

UUID: Identificador único universal.

Upstream: Datos que van del cliente al servidor.

Undervolting: Reducción del voltaje para disminuir el consumo energético.

V

Version Control: Sistema para gestionar cambios en código fuente.

VLAN: Red de área local virtual.

Volatile Memory: Memoria que pierde datos al apagarse.

Vector Graphics: Imágenes basadas en formas geométricas.

W

WebRTC: Tecnología para comunicación en tiempo real a través de la web.

Webhook: Método para recibir notificaciones automáticas de eventos.

Whitelist: Lista de elementos permitidos o seguros.

Wrapper: Interfaz que traduce comandos entre aplicaciones.

X

XOR: Operador lógico exclusivo (una o la otra, pero no ambas).

X.509: Estándar para certificados digitales y criptografía.

XQuery: Lenguaje de consulta para datos XML.

X11: Sistema de ventanas para interfaces gráficas en Unix/Linux.

Y

YubiKey: Dispositivo de autenticación multifactor.

YARN: Gestor de paquetes para JavaScript.

Yield: Palabra clave en lenguajes de programación para pausar y reanudar funciones.

Y2K38: Problema de representación de fechas en sistemas Unix en 2038.

Z

ZSH: Intérprete de comandos mejorado para Unix/Linux.

Zigbee: Protocolo inalámbrico de bajo consumo para IoT.

Zlib: Biblioteca de compresión de datos.
